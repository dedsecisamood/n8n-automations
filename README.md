# n8n Automation Workflows

A collection of expert-level n8n workflows for AI-powered content repurposing and lead management.

## Workflows

### 1. AI Content Repurposer

- **File:** `ai_content_repurposer.json`
- **Description:** Automatically repurposes long-form content (YouTube transcripts, blog posts) into social media formats like Instagram captions, Twitter threads, and LinkedIn posts.

### 2. Instagram Automation

- **File:** `instagram_automation_workflow.json`
- **Description:** Streamlines Instagram interactions and content scheduling.

### 3. WhatsApp AI Agent

- **File:** `whatsapp_ai_agent_workflow.json`
- **Description:** An intelligent WhatsApp assistant powered by AI to handle queries and information retrieval.

### 4. WhatsApp Lead Qualifier

- **File:** `whatsapp_lead_qualifier.json`
- **Description:** Automatically qualifies leads coming through WhatsApp based on predefined criteria.

## How to Import

1. Open your n8n instance.
2. Go to the **Workflows** tab.
3. Click on the **Import from File** button (top right).
4. Select the `.json` file you wish to import.
5. Configure your credentials for each node.
